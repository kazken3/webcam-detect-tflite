# webcam-detect-tflite

Example App detecting objects from video capture using TensorFlow Lite.

## Usage

```
$ ./webcam-detector
```

## Requirements

* OpenCV
* TensorFlow Lite

## Installation

After modifying Makefile

```
$ make
```
